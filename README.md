# RSA Project

A first-person environmental prototype created in Unity for the RSA Project Outcome assignment 
at the University of Winchester.

## About The Project

The player takes on the role of a hunter sent into a forest to eliminate a creature believed to be 
responsible for environmental destruction. As the kill the enemy, the player discovers that humanity is 
actually responsible for destroying the environment rather than the creature itself.

This prototype was developed individually over a twelve-week production period using Unity 6.3 LTS.

## Installation
### Download Build
1. Download the latest game build from the repository releases section.
2. Extract the ZIP folder.
3. Open the extracted folder.
4. Run the executable file: RSAProject.exe

### Clone Repository
git clone https://github.com/Mr-Wolfie616/RSA-Project.git

#### Open Project
1. Open Unity Hub
2. Click "Add Project From Disk"
3. Select the downloaded repository folder
4. Open using Unity 6.3 LTS

## Running The Project
1. Open the main scene located in: Assets/Scenes
2. Press the Play button inside Unity.
